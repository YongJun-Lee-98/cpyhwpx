cpyhwpx API Reference

한글(HWP) 자동화를 위한 C++ Python 확장 라이브러리

빠른 시작

import cpyhwpx

# HWP 객체 생성
hwp = cpyhwpx.Hwp()

# 문서 열기
hwp.open("example.hwp")

# 텍스트 삽입
hwp.insert_text("Hello, World!")

# 저장
hwp.save_as("output.hwp")
